# BRSocket-Node

Socket на nodejs. Принимающий события из очереди RabbitMQ. Авторизация происходит по технологии JWT.

## Установка

```
npm i -S brsocket-node
```

## Настройка

Установим PM2. Для стабильной работы нашего приложения.
```
npm i pm2 -g
```

Изменить файл env.js

## Использование
```
npm start
npm run stop
npm run restart
```
